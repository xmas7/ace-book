# Spadebook

[Spadebook Live](https://spbk.herokuapp.com/)

[Trello](https://trello.com/b/IN9o7iwZ/facebook-clone)

Spadebook is a 1:1 clone of Facebook. The MVP will feature user profiles, a news feed, friending, and wall posts.
